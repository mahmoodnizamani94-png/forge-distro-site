Under the Hood

forge-ota-lab: architecture 

  INPUT LAYER
  ─────────────────────────────────────────────────────
  payload.bin → PayloadParser
  .zip / .tar.gz → PackageExtractor → payload.bin

  CLASSIFICATION ENGINE
  ─────────────────────────────────────────────────────
  PayloadParser → PartitionManifest
  PartitionManifest → TierClassifier
    SUPPORTED    → direct extraction
    EXPERIMENTAL → extraction with integrity warnings
    FORENSIC     → hash capture only, no extraction

  OUTPUT LAYER
  ─────────────────────────────────────────────────────
  ExtractedImage → SHA256Verifier → /storage/OTA/
  All output: labeled [direct | reconstructed | partial]

  SECURITY MODEL
  No network. No cloud. Termux-compatible. GPL-3.0 licensed.

Install Guide

  1. 1

    Download the APK

    Tap the download button above or visit GitHub Releases.

  2. 2

    Allow installation

    Open Settings → Apps → Special app access → Install unknown apps and enable it for your browser.

  3. 3

    Install

    Open the downloaded APK file and tap "Install".

  4. 4

    Verify (optional)

    Compare the SHA-256 hash shown above with the output of:

    sha256sum /path/to/forge-ota-lab.apk

Device Requirements

Android Version
Android 10 (API 29) or higher
Architecture
arm64-v8a (64-bit ARM)
Storage
Sufficient free space for OTA package + extracted images
Permissions
Storage access for reading OTA files and saving extracted images
Root
Not required. Operates entirely in user-space
