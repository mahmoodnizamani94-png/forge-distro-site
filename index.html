Device Requirements

arm64-v8a (64-bit ARM)

Sufficient free space for OTA package + extracted images
Permissions
Storage access for reading OTA files and saving extracted images

Trust & Verification

Open Source

Full source code available on GitHub. Inspect every line, build from source, verify independently.

Verify Your Download

Every release includes a SHA-256 checksum. After downloading, verify with:

sha256sum forge-ota-lab.apk

Compare the output with the hash shown on this page.

Security Posture

No network calls at runtime. No cloud dependencies. No tracking. No account required. Your firmware data stays on your device — always.
